# Compilar y correr el proyecto
Para compilar y correr el servidor:
```bash
cd ./backend
npm install
npm run db:migrate
npm run dev
``` 

Para compilar y correr el cliente web:
```bash
cd ./frontend
npm install
npm run build
npm run dev
``` 
Y el cliente debería estar corriendo en la dirección http://localhost:3000
